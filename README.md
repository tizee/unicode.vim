# unicode.vim

Provide commands to convert Unicode value string to corresponding character.

```
" get the character of given hex unicode value
:<range>Unicode
:Unicode <hex-value>

" get hex unicode value of given string
:<range>GetUnicode
:GetUnicode <string>
```
